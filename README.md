# HLA Resolve
HLA typing from long reads

**Authors:** [Matthew Glasenapp](https://github.com/matthewglasenapp), [Alex Symons](https://github.com/FlyingFish800)

Usage
```
hla_resolve.py \
--input_bam INPUT_BAM \
--sample_name SAMPLE_NAME \
--platform {pacbio,ont} \
--output_dir OUTPUT_DIR \
--threads THREADS
```
